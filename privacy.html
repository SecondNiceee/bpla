1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей сайта uronu-net.ru (далее — «Сайт»).

Используя Сайт и предоставляя свои персональные данные, Пользователь выражает согласие с условиями настоящей Политики.

2. Оператор персональных данных

Контактные данные для связи: +7 (905) 335-01-09

3. Цели сбора персональных данных

Персональные данные Пользователя собираются и обрабатываются в следующих целях:

  • Идентификация Пользователя для оказания консультационных услуг
  • Связь с Пользователем для предоставления информации о страховых продуктах
  • Направление информационных и рекламных материалов (при наличии согласия)
  • Улучшение качества обслуживания и работы Сайта
  • Выполнение требований законодательства Российской Федерации

4. Перечень собираемых данных

Оператор может собирать следующие персональные данные:

  • Имя (фамилия, имя, отчество)
  • Контактный телефон
  • Адрес электронной почты
  • Сведения об объекте страхования (тип имущества)
  • Иные данные, предоставленные Пользователем добровольно

5. Правовые основания обработки

Обработка персональных данных осуществляется на следующих правовых основаниях:

  • Согласие субъекта персональных данных
  • Исполнение договора, стороной которого является субъект персональных данных
  • Исполнение обязанностей, возложенных на оператора законодательством РФ

6. Порядок обработки персональных данных

Обработка персональных данных включает сбор, запись, систематизацию, накопление, хранение, уточнение, извлечение, использование, передачу, обезличивание, блокирование, удаление и уничтожение данных.

Обработка осуществляется с использованием средств автоматизации и без таковых.

7. Защита персональных данных

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения:

  • Ограничение доступа к персональным данным
  • Использование защищённых каналов связи (SSL/TLS)
  • Контроль доступа к информационным системам
  • Регулярное обновление программного обеспечения

8. Передача персональных данных третьим лицам

Персональные данные могут быть переданы третьим лицам в следующих случаях:

  • С согласия Пользователя
  • Страховым компаниям для оформления страхового полиса
  • По требованию уполномоченных государственных органов в случаях, предусмотренных законодательством

9. Сроки хранения персональных данных

Персональные данные хранятся в течение срока, необходимого для достижения целей обработки, но не более 3 (трёх) лет с момента последнего взаимодействия с Пользователем, если иное не предусмотрено законодательством.
